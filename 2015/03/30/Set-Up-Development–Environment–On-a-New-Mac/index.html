Set Up Development Environment On a New Mac

Posted on 2015-03-30   |   In Mac   |  

In this tutorial you will set up the development environment on a new Macbook for some major languages like Python, Ruby, Java and Node environments. Even if you don’t program in all four, it is good to have them as many command-line tools use one of them.

System Update

First of all, update your system. For that: Apple Icon -> About This Mac -> Software Update

System Preferences

In Apple Icon -> System Preferences:

  • Trackpad ->
    • Tap to click
    • Secondary click -> Click in bottom right corner
  • Keyboard ->
    • Key Repeat -> Fast(all the way to the right)
    • Delay Until Repeat -> Short(all the way to the right)
  • Dock -> Automatically hide and show the Dock, and make the size of Icons small

In Battery Icon -> Show Percentage

The most popular package manager for OS X is Homebrew.

Install

An important dependency before Homebrew can work is the Command Line Tools for Xcode. These include compilers that will allow you to build things from source.

If you’re Iphone or Mac apps developer, you can install Xcode from the App Store or the Apple developer website.

iTerm2

Installation

Let’s install a better terminal than the default one. Download and install iTerm2.

Configuration

Open the Preference.

  • General:
    • Uncheck Open profiles window
  • Profiles:
    • Colors: I personally prefer the Solarized dark, and the cursor text and cursor color to yellow make it more visible.
    • Text: I choose 14pt Menlo.
    • Window: Adjust the transparency to 30%, and set the columns to 100, rows to 25.
  • Keys: Set hotkey to open and close the terminal to ‘command + option + \ ‘, or else you like.
